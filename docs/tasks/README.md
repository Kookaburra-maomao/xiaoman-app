## 📋 任务文档模板

创建新任务记录时，请使用以下模板：

```markdown
# 任务名称

## 任务时间
YYYY-MM-DD

## 任务目标
简要描述任务目标

## 执行步骤
1. 步骤1
2. 步骤2
...

## 完成情况
- ✅ 已完成项
- ⏳ 进行中
- ❌ 未完成

## 相关文件
列出修改的文件

## 技术要点
关键技术点说明

## 遇到的问题
问题描述和解决方案

## 注意事项
需要注意的问题或后续跟进事项

## 参考资料
相关文档链接
```

## 🔍 快速查找

### 按类型查找
- **功能开发**: 搜索 `FEATURE_` 或 `IMPLEMENT_`
- **Bug修复**: 搜索 `FIX_` 或 `BUG_`
- **重构**: 搜索 `REFACTOR_`
- **文档**: 搜索 `DOC_` 或 `CLEANUP_`
- **优化**: 搜索 `OPTIMIZE_` 或 `IMPROVE_`

### 按模块查找
- **认证**: 搜索 `AUTH_` 或 `JWT_`
- **日志**: 搜索 `LOG_` 或 `TRACKING_`
- **UI**: 搜索 `UI_` 或 `STYLE_`
- **API**: 搜索 `API_` 或 `SERVICE_`
