# 修复计划页面字体缩放问题

## 任务描述
禁用计划页面所有文字的系统字体缩放，确保 APP 布局不受系统字体大小设置影响。

## 问题背景
在 vivo 和 iPhone 设备上，当用户调整系统字体大小后，会影响 APP 的布局和排版。需要在所有 Text 和 TextInput 组件上添加 `allowFontScaling={false}` 属性。
