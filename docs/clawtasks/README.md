本目录用于记录由 Claw (OpenClaw AI Assistant) 接收和执行的任务，与 `docs/tasks/` 目录的区别：

## 📝 文件命名规范

```

```

- `YYYYMMDD` - 任务执行日期（8位数字）
- `TASK_NAME` - 任务名称（大写字母+下划线）
